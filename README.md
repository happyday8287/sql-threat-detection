# SQL Threat Detection Project

## Objective

Analyze login activity using SQL to identify suspicious behavior and potential security threats.

## What I Did

* Created a dataset of login activity
* Analyzed failed login attempts using SQL
* Identified an IP address with repeated login failures

## Result

* Detected suspicious IP: 10.0.0.50
* Observed multiple failed login attempts followed by a successful login
* This pattern may indicate a potential brute-force attack

## Tools Used

* SQL (SQLite)
* GitHub

## What I Learned

* How to analyze log data using SQL
* How to detect suspicious login patterns
* Basic understanding of threat detection
